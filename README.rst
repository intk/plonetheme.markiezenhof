============
 Markiezenhof theme
============
------------------------------
 An installable theme for Plone 5
------------------------------

`Latest version on PyPI <https://pypi.python.org/pypi/plonetheme.markiezenhof>`_


Documentation
=============
- Extends the theme `Modernbase <https://pypi.python.org/pypi/plonetheme.modernbase>`_


Installation
============================

Install plonetheme.markiezenhof by adding it to your buildout::

    [buildout]

    ...

    eggs =
        plonetheme.markiezenhof


and then running ``bin/buildout``
